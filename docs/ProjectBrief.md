# ElevateHer Learn - Project Brief

## Problem
Women trying to reskill for a job switch face four barriers:
1. No time flexibility
2. No job-readiness signal
3. Fear of asking doubts
4. No pause/resume design

## Solution
A micro-learning platform with:
- 3-7 minute snack lessons
- AI-generated content
- Job-readiness scoring
- Anonymous doubt wall
- Guilt-free pause/resume
- Job board integration

## Target Users
Women in India (Tier-2/3 cities) balancing caregiving + work, pursuing career switches into STEM roles.

## Tech Stack
- Frontend: React Native + Expo
- Backend: Firebase + Cloud Functions
- AI: OpenAI API
- Database: Firestore

## MVP Features
- Onboarding with life-anchored questions
- Dashboard with progress tracking
- AI-powered lessons and grading
- Job-readiness score
- Doubt wall with tiered answers
- Job board with diversity hiring
- Pause/resume learning
